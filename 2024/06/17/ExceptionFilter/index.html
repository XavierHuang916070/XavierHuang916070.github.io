aspdotnetcore

  • 何為Filter

  • 1 - Filter為Asp.Net Core MVC框架、Asp.Net Core Web Api框架或其他中間件中的組件，採用了切面程式設計機制(即: 會在專案中特定的程式碼執行處、特定的情況下自動執行自定義的程式碼，可想像成: 在特定的情況下”切入執行自定義的程式碼”)

  • 2 - Asp.Net Core中共有5種類型的Filter

  • (1) Authorization Filter

  • (2) Resource Filter

  • (3) Action Filter

  • (4) Exception Filter

  • (5) Result Filter

  • 3 - 所有的Filter都具有同步與非同步兩個版本，例如: IActionFilter介面、IAsyncActionFilter介面

  • Exception Filter

  • 1 - 當系統中出現未經處理的異常時，系統會自動執行Exception Filter
